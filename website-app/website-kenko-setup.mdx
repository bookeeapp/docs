## Domain \+ SSL Setup

To configure your domain:

1. Navigate to **SEO & Settings ‚Üí Site Domain**
2. Use a domain you own (e.g. `www.mystudio.com`)
3. Add the following DNS records:
   - `CNAME: www ‚Üí s.multiscreensite.com`
   - `A Record: @ ‚Üí IP Address provided by Kenko`
4. Generate SSL in **SEO & Settings ‚Üí Site SSL**
5. Enable "Force HTTPS"

<Warning>
  If you've changed DNS post-SSL generation, **regenerate** the certificate to avoid mismatch.
</Warning>
