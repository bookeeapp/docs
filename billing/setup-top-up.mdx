# Deep Dive into Top-Up Setup

Top-Up Packs are designed to add more credits to existing memberships, providing flexibility for users to increase their credit balance as needed.

## Creating a Top-Up Pack

You can create a Top-Up Pack in Kenko by following these steps:

<Note>
  Navigate to Sidebar > Setup > Pricing Options > Create new.
</Note>

Setting up Top-Up packs involves defining **Basic Details**, configuring **Credits**, and setting **Pricing Options**. In this guide, we'll walk you through each step.

## Basic Details

The first step is to define the **basic details** of your Top-Up Pack. Here, you will set the membership type, location, and related metadata.

<AccordionGroup>
  <Accordion title="Membership Type" icon="rotate">

    - **Top-Up Pack**: Add more credits to an existing membership.

    <Note>

  <Accordion title="Location & Plan Name" icon="location-pin">
    - **Location**: Select the location where the Top-Up Pack can be purchased (e.g., specific branch).

## Credits

In this step, you'll define the number of credits included in the Top-Up Pack and configure visibility.

<AccordionGroup>
  <Accordion title="Credits" icon="coins">
    - **Credits**: Define the number of credits that will be added to the membership. For example, 10 credits.
    - **Applicable on**: Choose whether the credits apply to all plans or selected plans.
    <div className="flex justify-center mt-4">
      <img
        src="images/Screenshot2025-04-23at7.34.28PM.png"
        alt="Membership Type"
        className="max-w-[600px] w-full rounded-lg shadow-sm"
      />
    </div>

    <div className="flex justify-center mt-4" />
  </Accordion>
  <Accordion title="Visibility" icon="eye">
    - **Visible to Customer**: Toggle visibility to customers on the app and website.
    - **Visible on POS**: Toggle visibility on POS in the CRM for staff to see.

    <div className="flex justify-center mt-4" />
  </Accordion>
</AccordionGroup>
