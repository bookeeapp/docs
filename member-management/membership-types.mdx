# Membership Types

Membership types define the service packages you offer to clients. Kenko allows you to create flexible membership structures that match your business model, from single-class passes to unlimited recurring memberships.

## Understanding Membership Types

Membership types in Kenko determine:

- **Access rights**: What services members can access
- **Pricing**: How much members pay and when
- **Duration**: How long the membership lasts
- **Benefits**: What special perks are included
- **Restrictions**: Any limitations on usage

## Creating Membership Types

<Steps>
  <Step title="Access Membership Settings">
    Navigate to **Admin Settings** > **Membership** > **Membership Types**.
  </Step>
  
  <Step title="Add New Membership">
    Click the **+ New Membership Type** button.
  </Step>
  
  <Step title="Enter Basic Details">
    Complete the required information:
    - Membership name
    - Description
    - Status (active/inactive)
    - Color tag (for visual identification)
  </Step>
  
  <Step title="Configure Pricing">
    Set the financial terms:
    - Price amount
    - Tax settings
    - Billing frequency (one-time, weekly, monthly, annual)
    - Free trial period (if applicable)
    - Joining fee (if applicable)
  </Step>
  
  <Step title="Set Duration and Renewal">
    Define the membership timeframe:
    - Duration type (unlimited, fixed period, class count)
    - Expiration settings
    - Auto-renewal options
    - Cancellation terms
  </Step>
  
  <Step title="Configure Access Rights">
    Specify what this membership can access:
    - Class types included
    - Number of visits allowed
    - Booking privileges (advance booking window)
    - Facility access
    - Resource reservations
  </Step>
  
  <Step title="Add Special Benefits">
    Include any additional perks:
    - Guest passes
    - Discounts on retail or services
    - Priority booking
    - Special event access
  </Step>
  
  <Step title="Set Availability">
    Determine who can purchase this membership:
    - Available to all or specific member groups
    - Promotional/limited time availability
    - Hidden/invite-only options
  </Step>
  
  <Step title="Save Membership Type">
    Click **Save** to create the membership type and make it available for sale.
  </Step>
</Steps>

## Managing Membership Restrictions

<AccordionGroup>
  <Accordion icon="calendar-week" title="Time Restrictions">
    Limit when members can attend:
    
    1. In the membership type settings, go to the **Restrictions** tab
    2. Select **Add Time Restriction**
    3. Choose days of week and time ranges
    4. Set whether restrictions apply to all or specific class types
    
    **Common example**: "Off-Peak Membership" only valid weekdays 9am-4pm
  </Accordion>
  
  <Accordion icon="map-marker-alt" title="Location Restrictions">
    For multi-location businesses, restrict access to specific locations:
    
    1. Go to the **Locations** tab
    2. Select applicable locations for this membership
    3. Choose "All Locations" or select specific ones
    
    **Common example**: "Single Club Membership" vs "Multi-Club Access"
  </Accordion>
  
  <Accordion icon="user-clock" title="Booking Restrictions">
    Manage how far in advance members can book:
    
    1. Go to the **Booking Privileges** section
    2. Set the advance booking window (e.g., 7 days)
    3. Configure maximum active bookings allowed
    
    **Common example**: Standard members book 7 days ahead, Premium members 14 days
  </Accordion>
</AccordionGroup>

## Pricing Strategies

Consider these effective pricing approaches for your memberships:

<CardGroup>
  <Card title="Tiered Pricing" icon="layer-group">
    Create good/better/best options that increase in price and value
    
    **Example**: Basic ($79/mo), Plus ($99/mo), Premium ($129/mo)
  </Card>
  
  <Card title="Commitment-Based Pricing" icon="handshake">
    Offer discounts for longer commitments
    
    **Example**: Month-to-month ($99/mo) vs. 12-month commitment ($79/mo)
  </Card>
  
  <Card title="Introductory Offers" icon="tag">
    Special first-month pricing to reduce signup friction
    
    **Example**: First month $49, then $99/month
  </Card>
</CardGroup>

## Managing Existing Membership Types

### Editing Memberships

You can modify existing membership types:

1. Go to **Admin Settings** > **Membership** > **Membership Types**
2. Find the membership type and click **Edit**
3. Make your changes and click **Save**

<Warning>
  Changes to existing membership types will not affect members currently enrolled in that membership. Price or term changes will only apply to new sign-ups unless you specifically migrate existing members.
</Warning>

### Archiving Memberships

When you no longer want to offer a membership:

1. Go to **Admin Settings** > **Membership** > **Membership Types**
2. Find the membership type
3. Click the **Archive** button
4. Confirm the archiving action

Archived memberships:
- Are no longer available for purchase
- Remain active for existing members until expiration
- Can be unarchived if needed
